## 1. The 2-torus as a compact abelian Lie group

**Definition.** Fix an integer $p \ge 2$. The **2-torus at modulus $p$** is

$$T^2 := (\mathbb{R} / p\mathbb{Z}) \times (\mathbb{R} / p\mathbb{Z}) \;\cong\; S^1 \times S^1.$$

Equipped with coordinatewise addition modulo $p$, $T^2$ is a compact abelian Lie group of dimension $2$. The Lie algebra is $\mathbb{R}^2$ and the exponential map $\exp : \mathbb{R}^2 \to T^2$ is the canonical projection $(\xi, \eta) \mapsto (\xi \bmod p, \eta \bmod p)$.

We use $T^2$ throughout; in the codebase the modulus rescaling appears in [teacher.py](../../../sobolev_distill_character/teacher.py) where `nodes_x = np.arange(p, dtype=np.float64)` lays out the lattice on $[0, p) \subset \mathbb{R}/p\mathbb{Z}$.

**Why the torus.** Modular addition $i + j \mod p$ is *periodic* in both arguments. Working on $\mathbb{R}^2$ (or $[0, p-1]^2$) requires manual identification of the boundary; working on $T^2$ has the identification built into the geometry. Every later property -- exact Fourier interpolation, winding-number invariants, the Dirichlet kernel as reproducing kernel -- depends on this choice.

## 2. The lattice of $p$-th roots of unity

**Definition.** The **lattice** is

$$\Lambda := \{(i, j) : i, j \in \{0, 1, \ldots, p-1\}\} \;\cong\; (\mathbb{Z}/p\mathbb{Z})^2 \;\subset\; T^2.$$

$\Lambda$ has $|\Lambda| = p^2$ points. It is a discrete subgroup of $T^2$; the quotient $T^2 / \Lambda$ is again a 2-torus (rescaled).

The lattice points are where the teacher's analytical value table is exact and where the cardinal interpolation nodes live. In the codebase, lattice points appear via `dataset.is_node` in [dataset.py](../../../sobolev_distill_character/dataset.py).

## 3. The fundamental character $\chi : \mathbb{Z}/p\mathbb{Z} \to S^1$

**Definition.** Let $\zeta := e^{2\pi i / p} \in S^1$, the primitive $p$-th root of unity. The **fundamental character** of $\mathbb{Z}/p\mathbb{Z}$ is

$$\chi : \mathbb{Z}/p\mathbb{Z} \;\longrightarrow\; S^1, \qquad \chi(s) = \zeta^s = e^{2\pi i s / p}.$$

**Proposition (group homomorphism).** $\chi(a + b) = \chi(a) \cdot \chi(b)$ for all $a, b \in \mathbb{Z}/p\mathbb{Z}$.

**Proof.** $\chi(a + b) = e^{2\pi i (a + b) / p} = e^{2\pi i a / p} e^{2\pi i b / p} = \chi(a) \chi(b)$. $\square$

This is the algebraic content of "modular addition becomes complex multiplication". Every later mechanism in the writeup -- the trig identity, the Fourier-features story, the helix manifold -- is a downstream consequence.

**Pontryagin duality (statement).** Let $G$ be a locally compact abelian group. The **Pontryagin dual** $\widehat{G}$ is the group of continuous group homomorphisms $G \to S^1$. For a finite cyclic group, $\widehat{\mathbb{Z}/p\mathbb{Z}} \cong \mathbb{Z}/p\mathbb{Z}$, generated by $\chi$. The other characters are $\chi^k(s) = \zeta^{ks}$ for $k = 0, 1, \ldots, p-1$. See Folland (2016, Chapter 4) for the general theory.

## 4. The teacher map $T : T^2 \to S^1$

**Definition.** The **teacher** is

$$T : T^2 \;\longrightarrow\; S^1, \qquad T(x, y) := \chi(x + y) = e^{2\pi i (x + y) / p}.$$

Here $x + y$ is the group multiplication on $T^2$ (which is the addition $\mathbb{R}/p\mathbb{Z}$, but to emphasise the group-theoretic content we call it multiplication when discussing characters).

$T$ factors as

$$T = \chi \circ \mu, \qquad \mu : T^2 \to T^1, \quad \mu(x, y) = x + y \mod p.$$

So $T$ is a smooth (in fact analytic) function $T^2 \to \mathbb{C}$ whose image lies on $S^1 \subset \mathbb{C}$. The image-on-$S^1$ property is what later motivates the unit-circle penalty $(|f|^2 - 1)^2$ in [losses.py](../../../sobolev_distill_character/losses.py).

In the codebase, the lattice values $T_{ij} = T(i, j)$ are computed in [teacher.py:build_character_teacher_mesh_periodic](../../../sobolev_distill_character/teacher.py) as `T_re[i, j] = cos(omega * (i + j))` and `T_im[i, j] = sin(omega * (i + j))` where `omega = 2 * pi / p`.

## 5. The fibration $T^2 \to S^1$

**Proposition (fibration).** $T : T^2 \to S^1$ is a surjective smooth submersion. Its fibres are the $p$ disjoint diagonals

$$T^{-1}(\zeta^c) \;=\; \{(x, y) \in T^2 : x + y \equiv c \pmod p\}, \qquad c = 0, 1, \ldots, p - 1,$$

each of which is a closed geodesic on $T^2$ of length $p\sqrt{2}$ in the flat metric.

**Proof sketch.** Surjectivity: $T(s, 0) = \chi(s)$ takes every value in the image of $\chi$, which is the full set $\{\zeta^c\}_{c=0}^{p-1}$. Continuity extends this to all of $S^1$ via density. The fibre over $\zeta^c$ is the preimage of the singleton $\{c\} \subset \mathbb{Z}/p\mathbb{Z}$ under $\mu$, which is the diagonal claimed. Each diagonal is a 1-d closed submanifold of $T^2$ (a geodesic in the flat metric inherited from $\mathbb{R}^2$). $\square$

**Remark (principal $S^1$-bundle).** Since $T^2 \to S^1$ admits a global section $s \mapsto (s, 0)$, the fibration is *trivial* as a principal $S^1$-bundle:

$$T^2 \;\cong\; S^1 \times S^1$$

where the first factor is the fibre and the second is the base. This is consistent with $\pi_1(T^2) = \mathbb{Z}^2$ being abelian (no non-trivial monodromy is possible).

## 6. Winding numbers along $\pi_1(T^2)$

**Proposition (winding $(1, 1)$).** Let $\gamma_1, \gamma_2 : [0, 1] \to T^2$ be the canonical generators of $\pi_1(T^2) = \mathbb{Z}^2$,

$$\gamma_1(t) = (pt, 0), \qquad \gamma_2(t) = (0, pt).$$

Then the induced map $T_* : \pi_1(T^2) \to \pi_1(S^1) = \mathbb{Z}$ sends $\gamma_1 \mapsto 1$ and $\gamma_2 \mapsto 1$. We say $T$ has *winding* $(1, 1)$.

**Proof.** Compute directly:

$$T(\gamma_1(t)) = T(pt, 0) = e^{2\pi i p t / p} = e^{2\pi i t}.$$

So as $t$ runs over $[0, 1]$, $T \circ \gamma_1$ winds exactly once around $S^1$ counterclockwise. Same calculation for $\gamma_2$. $\square$

This integer pair $(1, 1)$ is the *homotopy invariant* of $T$ as a map $T^2 \to S^1$. The set of such homotopy classes is

$$[T^2, S^1] \;\cong\; H^1(T^2; \mathbb{Z}) \;\cong\; \mathbb{Z}^2$$

and our teacher sits at the generator $(1, 1)$ of this group. See Hatcher (2002, Section 4.3) for the homotopy classification.

**Why this matters for the unit-circle penalty.** The penalty $(|f|^2 - 1)^2$ in [losses.py](../../../sobolev_distill_character/losses.py) constrains the student's image to lie on $S^1 \subset \mathbb{R}^2$. Continuous deformations of a map $T^2 \to S^1$ preserve the homotopy class. Therefore, *once the student is in homotopy class $(1, 1)$, the unit-circle penalty keeps it there*. Chapter 03 returns to this point.

## 7. The graph $\Gamma$ in $\mathbb{R}^4$

**Definition.** The **graph** of the teacher is

$$\Gamma := \{(x, y, \mathrm{Re}\,T(x,y), \mathrm{Im}\,T(x,y)) : (x, y) \in T^2\} \;\subset\; T^2 \times \mathbb{R}^2.$$

Since $T^2 \times \mathbb{R}^2$ has a canonical universal cover $\mathbb{R}^2 \times \mathbb{R}^2 = \mathbb{R}^4$, $\Gamma$ also pulls back to a 2-manifold in $\mathbb{R}^4$.

**Proposition.** $\Gamma$ is a smooth embedded 2-manifold in $T^2 \times \mathbb{R}^2$ diffeomorphic to $T^2$.

**Proof.** $\Gamma$ is the image of the smooth embedding $(x, y) \mapsto (x, y, \mathrm{Re}\,T, \mathrm{Im}\,T)$. The first two coordinates give a diffeomorphism with $T^2$. $\square$

This is the four-dimensional structure the user has in mind when describing the teacher space: input $(x, y) \in T^2$ has 2 dimensions; output $(\mathrm{Re}, \mathrm{Im}) \in \mathbb{R}^2$ has 2 dimensions; the *graph* is a 2-manifold inside the 4-d ambient. The "known set of minimums" is the lattice $\Lambda$ where the teacher's value, gradient, and Hessian are all available in closed form (chapter 02 gives the analytic formulas).

## 8. The lattice values: $p$-th roots of unity

The teacher's restriction to the lattice is

$$T|_\Lambda : \Lambda \to S^1, \qquad (i, j) \mapsto \zeta^{i + j \bmod p}.$$

The image of $T|_\Lambda$ is the $p$-element set

$$\{\zeta^0, \zeta^1, \zeta^2, \ldots, \zeta^{p-1}\} \;\subset\; S^1,$$

the **$p$-th roots of unity** evenly distributed around the unit circle. The preimage of $\zeta^c$ is exactly the diagonal $\{(i, j) \in \Lambda : i + j = c \bmod p\}$, which has $p$ elements.

So we may visualise the teacher restricted to the lattice as: $\Lambda$ has $p^2$ points; under $T|_\Lambda$ they sort into $p$ groups of $p$ points each, where the $c$-th group all map to the single value $\zeta^c$.

Decoding modular accuracy back to the group element happens in [train.py:_lattice_modular_truth_recovered](../../../sobolev_distill_character/train.py) via

$$\mathrm{recovered}(i, j) := \mathrm{round}\bigl(p \cdot \arg(f_\theta(i, j)) / 2\pi\bigr) \mod p.$$

Inverting $\chi$ via $\arg \circ \chi^{-1}$ is exactly reading off the discrete log of the predicted complex value to base $\zeta$.
